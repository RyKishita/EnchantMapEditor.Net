操作説明書

上部
・マップの縦マス数
・マップの横マス数
・パーツ1つ分の高さ ※1
・パーツ1つ分の幅 ※1
・「読込」パーツ画像の読み込み
・「Help」この説明を表示

※1 変更は「読込」ボタンでファイルを指定したときに反映されます。

マップ
・左クリック
    ・選択したパーツの配置
    ・衝突判定の配置
・左ドラッグで一括配置
    ・左ドラッグ中に右クリックで配置の取り消し
    ・衝突判定設定モード中の場合は範囲内の衝突判定を反転する。

レイヤーグループ
・レイヤー一覧
    ・チェックONで表示。OFFで非表示
    ・選択するとそのレイヤーを編集対象にする。
・「↑」選択レイヤーを前面に移動 ※2
・「↓」選択レイヤーを背面に移動 ※2
・「前景追加」前景レイヤーの追加
・「背景追加」背景レイヤーの追加
・「削除」選択レイヤーを削除

※2 前景/背景を越える移動はできない

パーツ配置グループ
・パーツ一覧
    ・左クリックでパーツの選択
・「空白」空白パーツの選択。マップをクリックするとクリア(透過)する。
・「塗りつぶし」
    ・ON：連続するマスを塗りつぶす
	・OFF：クリックまたはドラッグ範囲による配置
・「x?倍」パーツ一覧を拡大
・「パーツ」「衝突判定」
    ・パーツ：クリック箇所にマップパーツを置く
    ・衝突判定：クリック箇所に衝突判定を置く。再度クリックで解除
	    ・「取込」衝突判定データを取り込んで設定

下部
・「背景色」背景色変更
・「x?倍」マップを拡大
・「元に戻す」最後に行った操作を取り消して、前の状態に戻す。
・「やり直し」元に戻すボタンで取り消した操作をもう一度反映する。
・「作成」現在の入力値を元にコードを作成

動作環境
.Net Framework 4 (Client Profile)
添付のソースのビルドには Visual Studio 2010 または Visual C# 2010 Express が必要
